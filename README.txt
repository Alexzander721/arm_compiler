Экспорт слоёв
Чтобы подготовить слои (mif, shp) для АРМ таксатора нужно:
1) Создать проект в QGIS
2) Задать ему нужную систему координат (та система в которой находятся слои)
3) Запустить модуль ARM compiler
4) Выбрать слой полигонов-выделов
5) Выбрать атрибут в котором прописан № квартала
7) Указать целевую СК
6) Указать папку для сохранения выходных слоёв

Импорт слоёв
Импорт проходит по аналогичной процедуре, что и экспорт,
но здесь нам нужно выбрать слой линейных и поле в котором прописано его ID.

Внимание!
Программа использует ID Библиотеки формата ЕСАУЛ.
Для корректного отображения и работы эта библиотека должна быть установлена на устройстве.

В качестве маркеров для поиска слоёв используются следующие значения:
'ГИДР': 27, 'РУЧЬИ': 28, 'РУЧЕЙ': 28,'КАНАЛ': 29,'ЖЕЛЕЗН': 30, 'ЖД': 30, 'АВТО': 31,
'ГРУНТ': 31, 'ЗИМНИК': 32, 'ТРОПА': 33, 'ТРОПЫ': 33, 'ЛЕСН': 33, 'ЛЕСОВ': 33, 'КАНАВ': 34,
'ГРАНИЦ': 35, 'ПРОСЕК': 36, 'ЛЭП': 45,'ГАЗ': 46, 'ТЕЛЕФОН': 47, 'ЛИНИЯ': 47, 'СВЯЗЬ': 47, 
'СВЯЗИ': 47, 'МЕЛИОРАЦ': 48, 'ПОЖАР': 49, 'ПРОЧИЕ': 50, 'ВОДОПРОВОД': 56,  'НЕФТ': 82

Для того что бы программа смогла распознать имя слоя, называйте слои правильно.
Пример: слой "Гидрография" будет распознан, т.к. содержит маркер "ГИДР", а слой "Реки" нет.
